# 簡易全幅教學

我想在使用者一開始進入首頁(沒有任何parameter)的時候，顯示一個簡易的全幅輪播教學，讓使用者快速了解這個App的核心功能與價值，並引導他們開始使用。
每頁一個方形動畫與文字區塊，視螢幕landscape或portrait，自動調整布局。

## tutorial_pages 使用方式

這個目錄改採一頁一檔管理。

- 本檔用途：總索引、整體目標、共同規則
- 各頁用途：拆解單頁的文案、構圖、SVG 圖層、動畫時序與 MVP 範圍
- 實作順序：先 page 01，確認風格可行後再往後推

## 共同規則

1. 每頁都以「方形動畫區 + 文字區」組成。
2. 先以 SVG + CSS + emoji 完成，不使用 Lottie。
3. 單頁同時運動的主要元素盡量不超過 4 個，避免手機掉幀。
4. 核心形狀以 SVG 為主，emoji 用於事件節點、人物或情緒點綴。
5. 先做 MVP 動畫，再決定是否加入更細緻的進階效果。

## 以下是我對這個教學輪播的內容構想：

1. 忙得筋疲力竭，卻又覺得自己常常被打岔，好像沒有真正完成什麼重要的事？
2. 這是一個幫助我們回到人生主線的App，自律成了自然的事
3. 利用蕃茄時鐘法，搭配iPhone捷徑可以在開始任務時計時30分鐘，結束任務後計時10分鐘，使我們專注
4. 想法收集、設定任務、微任務、排程與中斷都在這軟體，節省APP間切換
5. 沒事多查看任務清單，一頁條列當下該做之事，輕鬆回到人生主線
6. 一切動作都紀錄在log，方便查詢
7. 可以與您自己的Google sheets串接，好備分您的資料，避免丟失、跨裝置同步與利用sheets 分析您的Log
8. 開始使用前，請先在 TaskPool 與 Scheduled 表中各新增項目，好停止這個教學
9. 這是Open source軟體，歡迎參與開發

## 實作細節

1. 如果進入首頁時 db.inbox 或 db.taskPool 或 db.scheduled 中沒有任何項目，就顯示這個教學輪播，然後有個按鈕可以收起這個教學輪播。
2. 先完成內容與基本樣式，再一頁一頁加入動畫效果。
3. 在每頁的方形動畫區塊中，可以使用 CSS 動畫或 Lottie 動畫來增加視覺吸引力。

### 各頁內容構想

- 頁1:
  * 靜態的一個黯淡的粗箭頭由畫面左邊到右邊，箭頭的終點有個旗子圖示，
  * 動畫則是箭頭上面有個圓形時鐘，然後有個扇形角度越來越大，一開始是黃色，然後過了中線後，接著另一個紅色扇形出現，角度越來越大，同時，一個黃色bar 慢慢由左往右填滿那個粗箭頭，填到一半，這部分就不再動
  * 然後有個☎️的圖示，很大的出現，縮小後放到剛剛停住的那個黃色方塊右邊，表示被電話打斷了
  * 然後一個較細的黃色線往下走，然後有個小人圖示出現，表示被打斷了
  * 然後線條由打斷的那個點繼續往右走，越來越快的有一堆的表示雜事與娛樂的圖示出現，表示被各種事情打斷了
  * 最後，走到代表亂七八糟的圖示

- 頁2:
  * 時鐘、箭頭、旗子都還在(依照頁1的初始狀態設定)
  * 前進(時鐘黃色扇形開始角度由0變大)、事件打岔、往下U型跑線、跑到U型底時 NBL 的圖示出現，這樣看起來就像是 NBL 是在幫助我們回到主線的路上(時鐘黃色扇形停止角度變大)
    * 這個步驟大概做兩次到三次，代表遇到兩到三次的打岔事件，
  * 此時，應該也接近最右邊的旗子了，代表接近目標了，繼續往右填滿直到填完
  * 最後，旗子變成彩色，有光輝，代表達成目標了
  * 圖片的底部可以有一個像書卷的東西，代表Log，NBL圖案出現，就加入一筆在Log圖上，表示都有在紀錄
