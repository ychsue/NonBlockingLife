# Resources 使用說明

## 概述

**Resources** 是外部參考資源庫，用於保存和管理您發現的有用的網址、教程、參考資料等。

## 主要功能

### 新增資源

點擊 **+ Add** 按鈕創建新的資源項目。系統會自動生成 Resource ID（以 `R` 為前缀）。

### 編輯資源

在桌面版本中，點擊表格行即可編輯；在移動版本中，點擊卡片即可打開編輯對話框。

### 搜尋資源

- 在搜尋框輸入關鍵字
- 可按 **Title、Category、Note、URL** 進行全文搜尋
- 支援兩種搜尋模式：
  - **OR 模式**（預設）：只要匹配任意一個關鍵字即可（例如搜尋 "React Vue" 會顯示包含 "React" **或** "Vue" 的項目）
  - **AND 模式**：必須匹配所有關鍵字（例如搜尋 "React Tutorial" 會顯示同時包含 "React" **和** "Tutorial" 的項目）

### 刪除資源

點擊表格右方的 **Delete** 按鈕刪除行項。

## 欄位說明

| 欄位 | 說明 |
|------|------|
| **Resource ID** | 系統自動生成，以 `R` 開頭（隱藏） |
| **Title** | 資源名稱 |
| **Category** | 分類標籤（例如：Tutorial、Reference、Tool） |
| **Received** | 資源添加時間 |
| **Note** | 備註欄位 |
| **URL** | 外部連結，可直接點擊「開啟」按鈕打開 |

## 最佳實踐

- 為每個資源賦予清晰的 **Title** 和 **Category**，便於後續搜尋
- **URL** 欄位支援直接跳轉到外部資源
- 使用 **Note** 欄位補充說明，例如為什麼保存這個資源
- 善用搜尋功能快速找到所需的資源

## 提示

- 移動版本會以卡片形式展示，桌面版本使用表格形式
- 搜尋是實時進行的，無需按 Enter 鍵
