# Task Pool

## 目的

1. 把那些需要花比較多時間的任務，比如說專案，放在這裡，讓使用者可以有個地方可以看到他們，並且決定要不要做。
2. 設定 `Daily Limit`，讓使用者可以設定每天在這些任務上面花多少時間，當超過這個時間的時候，
   1. 正常情況下他就會變成綠色，表示今天這個任務執行得很好。
   2. 如果 `Title` 含有 `⛔` 的話，表示這個任務其實是您不想花那麼多時間的(比如社群軟體)，所以，若使用太多，他會變成灰色外加刪除線，告訴您使用太多了。

## 使用方式

1. 新增：
   1. 點 `+Add` 就可以新增
2. 刪除：(不建議刪除，建議改為 Done 就好，這樣才有統計資料)
   1. 每一筆紀錄都有個 `Delete` 鈕，或者手機上向左滑也可
3. 編輯：
    1. 手機的話，條目向右滑進入編輯畫面，改完後按`保存`即可
    2. 電腦上的話，改動完，focus一離開即更新完畢
