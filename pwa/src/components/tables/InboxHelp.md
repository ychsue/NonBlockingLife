# Inbox

## 目的

將靈感快速記錄下來，日後再找時間分類

## 使用方式

1. 新增：
   1. 點 `+Add` 就可以新增
   2. iPhone 上可以將`捷徑`的 `NBL Inbox` 綁定到 `設定➡️輔助使用➡️觸控➡️背面輕點➡️點兩下` ，這樣，只要點兩下手機背面，就可以自動新增一筆想法
   3. iPhone 上任何網頁，在安裝 `NBL Inbox` 後，也可以分享該網址給 NonBlockingLife 的 Inbox
2. 刪除：
   1. 每一筆紀錄都有個 `Delete` 鈕，或者手機上向左滑也可
   2. 每天自己找時間分類，分類完就刪除
      1. 超小事一做完，就刪掉
      2. 小事而沒要求期限等時間問題的話，複製 title 後就改到 `Micro Tasks` 新增一筆
      3. 與時間相關的事情，就到 `Scheduled` 新增一筆
      4. 得花超過一天的任務的，就到 `Task Pool` 新增一筆
      5. 若是資料，請使用您習慣的外部紀錄資源的地方存放吧，如 `OneNote`, `Notion` 等等
3. 編輯：
    1. 手機的話，條目向右滑進入編輯畫面，改完後按`保存`即可
    2. 電腦上的話，改動完，focus一離開即更新完畢
