# Micro Tasks

## 目的

利用零碎時間處理的小任務，處理完就設為 Done，不讓它出現在 `Candidates` 裡面

## 使用方式

1. 新增：
   1. 點 `+Add` 就可以新增
   2. 由於頂多一天一筆，所以沒有對應的 iPhone 的 `捷徑`
2. 刪除：(不建議刪除，建議改為 Done 就好，這樣才有統計資料)
   1. 每一筆紀錄都有個 `Delete` 鈕，或者手機上向左滑也可
3. 編輯：
    1. 手機的話，條目向右滑進入編輯畫面，改完後按`保存`即可
    2. 電腦上的話，改動完，focus一離開即更新完畢
4. 開始與結束：
   1. 請由 `Candidates` 裡面選擇一筆任務開始，結束也在 `Candidates` 裡面完成，然後就會記錄到 `Log` 裡面了
